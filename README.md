# Obesity
R code and results for obesity paradox project  
The goal of this project is to analyze the effects of obesity status and CPC cell counts on cardiovascular outcomes. We first analyze the relation between the obesity status and CPC cell counts using linear regression models. We then construct Cox proportional hazard models to evaluate the effects of obesity status and CPC cell counts on cardiovascular outcomes.  

- Obesity.Rmd contains all the R code used for the data analysis.
- Obesity.pdf shows all the results, including tables and plots, generated by Obesity.Rmd.
